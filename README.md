Project Overview
This project presents an exploratory data analysis of Airbnb listings in the City of Buenos Aires. The objective was to identify the neighborhoods with the highest concentration of listings, analyze price distribution, and explore the relationship between location, number of listings, and price.
The project demonstrates skills in data cleaning, exploratory analysis, data visualization, and communication of insights through an interactive dashboard developed in Tableau.

Objectives

Identify neighborhoods with the highest number of Airbnb listings
Analyze price distribution across different neighborhoods
Explore the relationship between listing quantity and pricing
Provide a clear and interactive visualization for business insights

Dataset

The dataset contains publicly available Airbnb listings data for Buenos Aires, including:
Listing ID
Neighborhood
Price
Room type
Availability
Other listing attributes

Data Preparation

The following cleaning and preparation steps were performed:
Standardization of price format
Data type corrections
Removal of inconsistent or missing values
Selection of relevant variables for analysis
Creation of calculated metrics such as listing counts and median price

Tools Used

Tableau Public
Microsoft Excel / Google Sheets
Exploratory Data Analysis techniques

Dashboard Features

The interactive Tableau dashboard includes:
Bar chart showing neighborhoods with the highest number of listings
Geographic map highlighting price distribution
Scatter plot exploring the relationship between listing quantity and price
Clear visualization of key metrics

Key Insights

Palermo and Recoleta present the highest concentration of listings
Higher median prices tend to appear in central and high-demand areas
There is no direct linear relationship between listing quantity and price
